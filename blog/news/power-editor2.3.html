Introducing Power Editor v2.3!
New update is created for you, new features and bug fixes upcoming.


Drag & drop support added

This update added support for the Drag & drop function,
thanks to which you can quickly open any file by simply dragging
it to the top taskbar or tab bar.


Updated Notepad in Power Editor app

In Power Editor version 2.3 the notepad has been updated.
I've updated the interface to comply with Fluent Design regulations
and added a few new features. For example, scaling text, using the
zoom panel, also now the text can be scaled with the mouse wheel, after
holding down the Ctrl key on the keyboard. In the future I've also added integration
with Microsoft 365 apps like OneNote, To Do, and voice input using Cortana.
The Bing search engine is already integrated into Notepad.


Updated files extension for Power Editor

I also didn't forget about the built-in extension for viewing folders and
quickly opening files directly from the editor. In update 2.3, the application interface
was slightly redesigned and a bug was fixed with opening a folder in a new window.


Home page new features and personalization

The update also affected the home page of the application.
Now you can pin and unpin Quick Actions to the home page. The interface of
fixed projects and files has been redesigned. In the personalization settings, a custom wallpaper
section for the application has been added.


Power Editor also supported on Xbox consoles!

Power Editor can also be used on Xbox consoles to edit or create text files,
as well as work with source code. Supported also on HoloLens, Surface Hub and Windows
for ARM, because this project based on UWP and use WinUI controls library. I want to warn
anyone who wants to use Power Editor on Xbox consoles. On Xbox, there are some restrictions about access
to the file system, so most likely you will open files through the command to open a file, and save through the save dialog.
Features such as share or files, favorites or recent will not work on Xbox.


Bug fixes and stability improvements

I've also fixed a lot of minor bugs and improved the stability of the app.
I want to say thank you to everyone who supports me on Twitter and to those who
use Power Editor app for Windows. ☺️


For Microsoft

Power Editor app based on UWP and WinUI library.

I don't think UWP should be abandoned, as it's likely the future of Windows development.

Why must have attention to UWP?
UWP should be given attention because this platform is a quick
replacement for desktop Windows applications, as UWP solutions are more flexible and versatile. By the same principle,
Windows should work, it should be available to everyone and on every device, and not just on a PC. It is not necessary
to position the system only for the PC, it is necessary to expand the horizons and move on. Windows should not only be functional
and convenient, but also mobile, since most people prefer smartphones to solve simple tasks. Bringing the new Windows to the
smartphone is what Microsoft needs to do in 2022. We need to make Windows more versatile and flexible than it is now. This will expand
the number of form factors for Windows-based devices. (TVs, PCs, Xbox, Tablets, Phablets, Mobile devices and IoT devices).
