Introducing Power Widgets version 1.5!

01/27/2022 11:30 AM

What do you need to know?

Mini-apps are replacements for old live tiles. We make sure that you can access important information for you without being distracted by launching applications. You can now interact with the Outlook widget and reply to mail or create calendar events with just a few clicks right from the widget panel.

What's new?

Design improvements

As you can see, the new widgets have been significantly redesigned in terms of the user interface. For example, the title and icon of the widget are now separated from the main content, and new control buttons and widget settings have been added. This greatly simplifies interaction with widgets. Updated and redesigned weather widget also get fluent design clean look.



Fast actions

Now, when you click on the network control button, you can quickly turn on or turn off Wi-Fi, see the available connections, as well as the network speed. When you press the Bluetooth button, you can see a list of connected devices, as well as their battery charge. You can also use fast actions to quickly access various system settings. You can add your custom fust actions. For me, it's really cooolll feature.

Outlook widget improvements

In this major 1.5 update, we also fixed bugs related to Outlook mail and added the ability to share a pinned contact. It also became possible to quickly delete an event from the calendar and view the event associated with a specific date.

Also added:
- New mail view dialog;
- Local events;
- Update interval;
- Updated design;

Redesigned settings

Options now open in a widget pop-up dialog, making them useful on any type of device.

Quick access to widgets using Application commands. (Coming soon)

Right-clicking on the widget bar icon in the Windows taskbar will open the gadget launch menu. It is very convenient that the gadget runs on top of all windows if you run it this way.

New sharing features.(Coming soon)

Windows Share will now be able to open links in the Edge browser or quickly search selected text.
You can easily pin the site you want to the widget ribbon using the share feature in Microsoft Edge.
SO, I think it's really cooolll feature!

Start menu, resize and replace widgets feature.

Now you can resize the widget or move it. Several resizing options are available.
• Small
• Medium
• Large

You can also change the position of the widgets. Depending on the size of the widget, the information on it may change.

New command bar

The 1.5 update added a new command bar. Thanks to which it has become even easier to manage widgets. Also in the command bar there are quick actions and a settings button with a Microsoft account. The command bar also has something like a "start" button that allows you to add or remove, as well as move widgets.

Full screen mode and new personalization features.

An equally interesting feature is the ability to personalize the widget feed. Now, starting from version 1.5, you can set the background image of the widget ribbon, adjust the blur effects and change the theme color. It also became possible to set your own (custom) image.

Microsoft Edge widget(Preview).

Introducing the Microsoft Edge Widget in Power Widgets! Now you can quickly and easily search the web using the search bar in the Microsoft Edge widget right in the widget bar. The search bar in the widget supports interactive search.

What is interactive search?

Interactive search is designed for quick search on the Internet, or rather in specific resources. You can quickly switch between search engines and resources where you should find something.

Supported places for searching(at the moment):
• Bing
• Google
• DuckDuckGo
• Yahoo
• Wikipedia
• YouTube
• Spotify
• Peoples
• Maps
• Microsoft Store

Pin your favorite sites to Microsoft Edge widget.

You can pin your favorite sites directly to the widget panel using the Microsoft Edge widget. Just open the desired site in the Edge browser and click the share button, then select Pin to Power Widgets, your site will be pinned to the Microsoft Edge widget. You can also use the quick site pinning button and pin the site without opening a browser.

Microsoft Edge widget search history.

You can view or clear your search history by switching to the History tab in the Microsoft Edge widget.

Personalization

Now you can customize the widgets feed for yourself by choosing the proposed wallpaper or applying your own, you can also change the accent color of the theme in the application or adjust the transparency and blur mode.

Free mode for widgets

You can run widgets in free mode. In this form, the widget that you run in free mode will be launched on top of all windows and you can interact with it separately from the main widgets feed, this is like picture-in-picture mode, but for widgets only.

Context commands for widget feed launcher icon on taskbar.(Coming soon)

Well, if for some reason you do not want to open or switch to the widget ribbon, just right-click on the Power Widgets application icon and you can launch the widget you need on top of all windows. (Free mode)

Conclusion

In general, update 1.5 is not bad, but there are a few more features that will come to us already on 1.5.1. The next update, namely 1.5.1, will be small and should generally be aimed at improving performance and adding small functionality. Let it be the secret of which one.
